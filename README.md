##  Project Overview
This project automates the process of network access requests using ServiceNow. 
Employees can request network access through a catalog item, which is automatically 
approved, assigned, and tracked using Flow Designer and task automation.

3. Task is created automatically
4. Assigned to Network Team
5. Status updates are tracked
